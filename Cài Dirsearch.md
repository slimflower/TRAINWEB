# Cài dirsearch 
## 1.Cài đặt Python (nếu chưa có):
- Yêu cầu Python 3.7 hoặc cao hơn.
- Cài đặt Python bằng lệnh:

         sudo apt update && sudo apt install python3 python3-pip -y

## 2.Clone Dirsearch từ GitHub:

       git clone https://github.com/maurosoria/dirsearch.git
         cd dirsearch

## 3. Cài đặt các thư viện cần thiết 

         pip install -r requirements.txt
# Sử dụng dirsearch
## Lệnh cơ bản 
        python3 dirsearch.py -u <URL>
## Sử dụng từ điển tuỳ chỉnh 

         python3 dirsearch.py -u http://example.com -w /path/to/wordlist.txt
## Quét đa luồng 
        python3 dirsearch.py -u http://example.com -t 20
